Privacy-first analysis

Your text stays in your browser

The current version of this tool uses client-side keyword detection only. There is no backend, no database, and no external API call used in analysis.

That means your pasted policy text is processed directly in your browser session.

  • No account required to run a check.
  • No third-party API receives your policy text.
  • No uploaded files or stored terms history.
  • Simple, transparent keyword scoring logic you can inspect in code.

If you decide to extend this app later with cloud features, keep this page updated so users understand exactly how their data is handled.
